# Weather Forecast App

## Overview
This project is a weather forecast web application that allows users to search for a location and view its current weather data. The application fetches weather information from the Visual Crossing API and displays it in a simple interface.

The goal of this project was to practice working with APIs, asynchronous JavaScript, and using Webpack as a module bundler.

## Features
- Search for weather by location
- Fetch weather data from the Visual Crossing API
- Display relevant weather information
- Toggle temperature between Celsius and Fahrenheit
- Dynamic UI changes based on weather conditions

## Technologies
- JavaScript (ES6+)
- Webpack
- HTML
- CSS
- Visual Crossing Weather API

## Concepts Practiced
- Fetching and processing API data
- async/await and Promises
- Modular JavaScript with Webpack
- DOM manipulation and event handling

## Purpose
The purpose of this project was to practice working with external APIs and organizing JavaScript applications using Webpack.
